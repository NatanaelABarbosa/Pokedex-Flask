# Pokédex Flask

Projeto desenvolvido para a disciplina da faculdade utilizando Python e Flask. A aplicação consome dados da PokéAPI para exibir informações sobre Pokémon de forma simples e intuitiva.

## Funcionalidades

- Buscar Pokémon pelo nome.
- Exibir informações básicas dos Pokémon.
- Visualizar imagem, tipos e atributos.
- Interface web desenvolvida com Flask.

## Tecnologias Utilizadas

- Python
- Flask
- HTML
- CSS
- PokéAPI

## Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/NatanaelABarbosa/Pokedex-Flask.git
```

2. Acesse a pasta do projeto:

```bash
cd Pokedex-Flask
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Execute a aplicação:

```bash
python app.py
```

5. Abra o navegador e acesse:

```text
http://localhost:5000
```

## Objetivo

O objetivo deste projeto é praticar o desenvolvimento de aplicações web utilizando Flask, integração com APIs externas e manipulação de dados em Python.
